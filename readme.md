# Ejercicio N°4 Gestión de Paquetes

Servidor básico creado con Node.js y Express que muestra,fecha y hora actual formateada con dayjs. Tambien muestra en consola mensajes en consola con colores usando chalk

 # Tecnologias
 - Node.js.
 - Express.
 - days.
 - chalk.

 # Funcionalidades
- Muestra un mensaje de bienvenida en color verde en consola.
- Muestra la fecha y hora actual en color amarillo.
- Devuelve la fecha actual al acceder a la ruta principal /

        
# Instalación

1. Clonar el repositorio o descargar el proyecto.
2. Instalar dependencias: **"npm install"**
3. Ejecutar **" npm start"**
